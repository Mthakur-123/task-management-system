##  Task Management System

A simple and efficient Task Management System built using Java and Spring Boot to manage daily tasks with ease. This application allows users to create, update, delete, and track tasks.


##  Features

- Create new tasks  
- Update existing tasks  
- Delete tasks  
- View all tasks  
- RESTful API implementation  
- Basic authentication using Spring Security  

## 🛠️ Tech Stack

- Core Java  
- Spring Boot  
- Spring Data JPA  
- Hibernate  
- MySQL  
- Spring Security  
- Maven 



## ⚙️ API Endpoints

| Method | Endpoint        | Description        |
|--------|----------------|--------------------|
| GET    | /tasks         | Get all tasks      |
| GET    | /tasks/{id}    | Get task by ID     |
| POST   | /tasks         | Create new task    |
| PUT    | /tasks/{id}    | Update task        |
| DELETE | /tasks/{id}    | Delete task        |
